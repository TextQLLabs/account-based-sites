Logo Files for Mastercard CIO Website
=====================================

Place the following logo files in this directory:

1. textql-logo.png - Your company logo
2. mastercard-logo.png - Mastercard logo

Logo Requirements:
- File formats: PNG with transparency preferred
- Resolution: High resolution (at least 240px wide)
- Height: Both logos will be displayed at a height of 40px in the website footer

Example file path: 
/logos/textql-logo.png
/logos/mastercard-logo.png
